# Controle-de-Estoque-V0.4
> save data in database PostgreSQL

### Libraries:
- keyboard
- sqlalchemy
- pandas

### How to Start: 
- Run **setup.py** to create a database and a table on postgreSQL.
- Run **main.py**.

### Installers:
- `pip install SQLAlchemy`
- `pip install pandas` 
- `pip install keyboard`
