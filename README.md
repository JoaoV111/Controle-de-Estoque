# Controle-de-Estoque-V0.4
> Controle de estoque com armazenamento em servidor PostgreSQL

### Libraries:
- keyboard
- sqlalchemy
- pandas

### How to Start: 
- run setup.py to create a database and a table on postgreSQL.
- then run main.py

### Installers:
- `pip install SQLAlchemy`
- `pip install pandas` 
- `pip install keyboard`
